## 🎯 **How to Use New Features**

### 🎨 **Customize Theme**
1. Go to **Customize** tab in navigation
2. Select **Themes** tab
3. Choose a preset or create custom
4. See changes instantly!

### 📱 **Rearrange Layout**
1. Go to **Customize** → **Layout**
2. Drag modules to new positions
3. Drop to rearrange
4. Changes save automatically

### 🖼️ **Add Background**
1. Go to **Customize** → **Background**
2. Click **Add Image**
3. Select your image file
4. Adjust opacity if needed
